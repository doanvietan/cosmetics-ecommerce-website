# Cosmetics E-Commerce Website

## Introduction
This project is an online cosmetics shopping website developed using Java Servlet, JSP, and MySQL.

## Features
- User login/register
- Product management
- Shopping cart
- Order management
- Admin dashboard

## Technologies Used
- Java Servlet
- JSP
- MySQL
- Bootstrap
- Apache Tomcat

## Database
Database name: 'cosmetics_db'
